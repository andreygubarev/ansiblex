# `getansible.sh` - Portable Ansible

## Usage

```bash
# execute without installing
curl -s https://getansible.sh/ | sh -s -- ansible-playbook playbook.yml

# install and execute
curl -s https://getansible.sh/ | sh
getansible.sh -- ansible-playbook playbook.yml
```
